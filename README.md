# 🏥 藥局自動排班系統

## 系統簡介
符合台灣勞基法四週變形工時的藥局自動排班系統，純HTML/CSS/JS開發，無需後端。

## 功能特色
✅ 自動合法排班引擎
✅ 台灣勞基法合規檢查
✅ 拖放式編輯介面
✅ 手機優先設計
✅ PWA支援（可安裝到主畫面）
✅ 離線可用
✅ 資料本地儲存

## 技術架構
- 前端：HTML5 + CSS3 + JavaScript (ES6+)
- 儲存：LocalStorage
- 部署：Vercel (靜態網站)
- PWA：Service Worker + Manifest

## 使用說明
1. 載入員工資料
2. 設定排班期間
3. 點擊「自動排班」
4. 手動調整（可選）
5. 匯出使用

## 法律合規
- 四週總工時 ≤ 160小時
- 每週工時 ≤ 40小時（四週平均）
- 連續工作 ≤ 6天
- 每7日 ≥ 1天休息
- 班次間隔 ≥ 11小時
